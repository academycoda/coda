# Project Overview

Coda is a Laravel + Livewire web application for a local programming academy focused on modern web development and AI driven software creation. The application serves as a public-facing website that presents the academy, its mission, courses, and educational philosophy, while also allowing students to browse course details and submit applications online. The platform should be designed as a clean, modern, and extensible system that supports both the presentation of the academy as a brand and the practical workflow around course promotion, student onboarding, and application management.

Language: Slovak
